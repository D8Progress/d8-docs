> **Note:** Quick Capture features are available starting in version 0.8.

ADHD brains have thoughts that arrive fast and leave faster. Quick Capture helps you catch them before they disappear.

## Quick Notes

The fastest way to capture a thought. Open the app, type, done.

### Creating a Quick Note

**From anywhere in the app:**

1. Press **Ctrl+N** (or **Cmd+N** on Mac)
2. Type your thought
3. Press **Enter**

That's it. The note is saved and you can get back to what you were doing.

**From the Quick Notes page:**

1. Click the text area
2. Type your note
3. Click **Save** or press **Enter**

### Note Features

**Automatic Link Detection**

If your note contains URLs, they're automatically extracted and saved as clickable links. Write:

> "Check out https://example.com for vacation ideas"

And the link becomes clickable in your note.

**Tags**

Add hashtags to organize notes:

> "Buy groceries #shopping #errands"

Tags are extracted and make notes searchable.

**Folders**

Organize notes into folders:

- Inbox (default)
- Ideas
- Work
- Personal
- Custom folders you create

### Converting Notes

The real power is turning notes into action. Convert a note to:

| Convert To   | What Happens                         |
| ------------ | ------------------------------------ |
| Task         | Creates a task from the note content |
| Goal         | Creates a goal from the note         |
| List Item    | Adds to an existing quick list       |
| Contact Note | Attaches to a contact's profile      |
| Gift Idea    | Saves as a gift idea for a contact   |

**To convert a note:**

1. Open the note
2. Click **Convert**
3. Choose the type
4. Select destination (which list, which contact, etc.)
5. Confirm

The note moves to "Converted" status so you know it's been processed.

### Note Statuses

| Status    | Meaning                         |
| --------- | ------------------------------- |
| Inbox     | New, unprocessed note           |
| Pinned    | Important, stay visible         |
| Converted | Turned into something else      |
| Archived  | Done with, but kept for history |

### Auto-Reset Lists

Perfect for recurring checklists!

**Daily Reset:**
Every day, all items uncheck automatically. Great for:

- Daily medication checklist
- Morning routine
- End-of-day shutdown

**Weekly Reset:**
Every week (on a day you choose), all items uncheck. Great for:

- Weekly cleaning checklist
- Meal prep list
- Weekly review items

**On Complete:**
When all items are checked, they all uncheck. Great for:

- Shopping lists (reset after shopping trip)
- Packing lists (reset after unpacking)

### Templates

Create reusable templates for lists you make often.

**Creating a Template:**

1. Create a list with all the items you want

**Using a Template:**

1. Click **+ New List**
2. Click **From Template**
3. Choose your template
4. Customize the name if needed
5. Your new list has all the template items!

## Tips for ADHD Brains

### Keep the Inbox Empty

Process your quick notes regularly. Don't let them pile up! Schedule 5 minutes daily to:

- Convert notes to tasks
- Archive notes you don't need
- Pin notes you want to keep visible

### One List Per Context

Instead of one giant list, make separate lists:

- **Grocery - Costco** (bulk items)
- **Grocery - Target** (regular items)
- **Hardware Store**
- **Online Shopping**

### Use Auto-Reset

If you have recurring checklists, auto-reset saves time. You never have to manually uncheck everything.

### Pin Sparingly

If everything is pinned, nothing stands out. Keep only 3-5 notes pinned at most.

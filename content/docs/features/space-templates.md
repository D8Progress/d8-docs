> **Note:** Communication features are available starting in version 0.9.

Space templates let you save a space's structure and reuse it later. Create your perfect channel organization once, then spin up new spaces in seconds.

## Built-in Templates

Deliber8 Progress includes templates for common use cases.

### Family Space

Perfect for household coordination.

**Channels included:**

| Category   | Channels                       |
| ---------- | ------------------------------ |
| General    | #announcements, #general       |
| Planning   | #calendar, #shopping, #meals   |
| Health     | #health-updates, #appointments |
| Activities | #outings, #hobbies             |

**Default settings:**

- Visibility: Private
- #announcements set as announcement channel

**Best for:** Families coordinating daily life, schedules, and activities.

### Friend Group

Designed for casual social coordination.

**Channels included:**

| Category  | Channels                    |
| --------- | --------------------------- |
| Hangout   | #general, #memes, #random   |
| Interests | #gaming, #movies-tv, #music |
| Planning  | #events, #trips             |

**Default settings:**

- Visibility: Private
- All channels are open for discussion

**Best for:** Friend groups planning hangouts, sharing interests, and staying connected.

### Blank

Start with an empty slate.

**Channels included:** None

**Best for:** Custom spaces where you know exactly what you want.

## Using a Template

1. Click **+ New Space** in the sidebar
2. Select a template:
   - **Family Space**
   - **Friend Group**
   - **Blank**
   - Or choose from **Your Templates**
3. Click **Continue**
4. Customize the space:
   - Name (required)
   - Description (optional)
   - Icon
   - Visibility
5. Click **Create Space**

All channels and categories from the template are created automatically.

## Creating Custom Templates

Save your perfect space setup for future use.

### Save Current Space as Template

1. Open the space you want to save
2. Click **Settings**
3. Click **Save as Template**
4. Enter:
   - **Template Name** ‚Äî Something descriptive
   - **Description** ‚Äî What this template is for
5. Click **Save**

### What Gets Saved

| Saved                 | Not Saved             |
| --------------------- | --------------------- |
| Channel names         | Messages              |
| Channel types         | Members               |
| Channel descriptions  | Pinned messages       |
| Categories            | Linked entities       |
| Category organization | Channel permissions\* |
| Default visibility    | Custom role settings  |

\*Basic channel types (text, announcement) are preserved.

## Managing Templates

### View Your Templates

1. Click **+ New Space**
2. Scroll to **Your Templates**
3. See all saved templates

### Delete a Template

2. Find the template in **Your Templates**

### Edit a Template

Templates can't be edited directly. Instead:

1. Create a new space from the template
2. Modify the channels and categories
3. Save as a new template
4. Delete the old template

## Best Practices

### Start with Built-in Templates

The Family and Friend Group templates cover most common needs. Start there and customize.

### Don't Over-Engineer

New spaces don't need 20 channels. Start with:

- #general
- #announcements
- 2-3 topic channels

Add more as needed.

### Name Templates Clearly

Good names:

- "Work - Client Project"
- "Personal - Side Projects"
- "Family - Nuclear"

Vague names:

- "Template 1"
- "New template"
- "Copy of Space"

### Review Before Saving

Before saving a space as a template:

1. Remove one-off channels
2. Clear descriptions of temporary content
3. Ensure categories make sense generically

## Tips for ADHD Brains

### Templates Reduce Decision Fatigue

Instead of deciding channel structure every time, just pick a template. Done.

### Create a "Minimal" Template

For when you want to start a space but can't decide on structure:

- #general
- #random

That's it. Add more later.

### Save As You Go

When you create a space structure you love, save it immediately. Future you will thank past you.

### Use Templates for Consistency

If you have multiple similar spaces (work projects, client teams), using the same template keeps them consistent. Less mental overhead switching between them.
